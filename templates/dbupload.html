Formatting

  1. Make sure you disable automatic list detection. On Google Docs this is Tools >> Preferences >> Automatic List Detection
  2. Start with 1. "First Question", make sure to have a space after the period for every question
  3. Use an empty line after question to signify its end
  4. Multiple Choice Question choices are to go under the question, with no empty lines in between
  5. Denote the correct multiple choice answer by a * in front of the letter
  6. In Free Response Questions, you may surround some variables in brackets [] to randomize those values in the tests
